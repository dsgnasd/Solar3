Инструкция

Как работать с Solar3

1
Подключи токены
Один файл — весь дизайн.
<link rel="stylesheet" href="tokens.css" />
2
Прочитай спек перед компонентом
В specs/ лежат размеры, состояния, правила. Не изобретай — проверь.
3
Только токены, без хардкода
Нужно добавить новое значение? Сначала проверь — есть ли токен. Если нет — добавь в tokens.css.
4
Проверь аудитом
После правок запусти скрипт — он найдёт все хардкод-значения.
node scripts/token-audit.js
